# Единицы измерения

## transform

1.  Чтобы трансформации вдоль оси z работали и выглядели максимально естественно, трансформируемый элемент должен лежать в родителе, которому задано свойство `perspective`:

```css
.parent {
  perspective: 500px;
}

.child {
  transform: translateZ(100px) rotateX(25deg);
}
```

2.  Если свойство transform имеет значение, отличное от none, то создаётся новый контекст наложения. Это означает, что относительно этого элемента теперь будут позиционироваться все дочерние элементы, у которых `position: fixed` или `position: absolute`
